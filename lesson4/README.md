# lesson 4: 循环

## while循环

```python
i = 0
while i < 100:
    i = i + 1
```

```python
i = 0
while i < 100:
    i = i + 1
else:
    print("i >= 100")
```

## 列表（list）

列表是一个包含了多个变量的变量

### 初始化

1. `l = []`
2. `l = list()`

### 列表的基本操作

- 向列表末尾添加一个元素
  - `l.push(element)`
- 删除列表最后一个元素
  - `l.pop()`
